## 🌱 Areas of Interest

- **Data Engineering** — 다양한 데이터를 수집·정제·저장하고, 안정적인 데이터 흐름을 구성하는 과정
- **Data Science** — 데이터를 분석하고 시각화하며, 의미 있는 패턴과 인사이트를 찾는 과정
- **Databases & Data Pipelines** — 데이터베이스 설계와 ETL 파이프라인을 통해 데이터를 관리하는 방법
- **AI Agents & RAG** — 데이터와 지식 정보를 활용해 작업을 보조하는 Agent 기반 시스템
- **Workflow Automation** — 데이터를 바탕으로 반복적인 업무를 보조하고 효율화하는 방법

## 📖 Learning Step by Step

> 작은 프로젝트를 통해 데이터가 수집되고, 저장되고, 분석과 서비스에 활용되는 과정을 익히고 있습니다.

- **Data Collection & ETL**  
  API, 웹 스크래핑, 파일, 공개 데이터 등 다양한 소스에서 데이터를 수집하고 정제·저장하는 과정

- **Database & Data Modeling**  
  관계형·비관계형 데이터베이스를 활용해 데이터를 구조화하고 관리하는 방법

- **Data Analysis & Visualization**  
  Python 기반 도구로 데이터를 탐색·분석하고 시각화하는 과정

- **Backend & API**  
  API를 통해 데이터베이스와 애플리케이션을 연결하고, 데이터를 활용하는 기능을 구현하는 방법

- **AI Agents & RAG**  
  검색, 데이터 조회, 요약 등의 작업을 연결해 데이터 활용을 돕는 Agent 흐름

- **Responsible Automation**  
  보안, 권한, 검토 절차를 고려하며 사람의 판단을 보조하는 자동화 방식

## 🧩 Small Projects

- **Data Collection Pipeline**  
  API, 웹 스크래핑, 파일 등 여러 데이터 소스에서 데이터를 수집하고 정리하는 흐름을 구현하며 학습하고 있습니다.

- **Data Analysis Practice**  
  수집한 데이터를 정리·분석하고, 시각화를 통해 패턴을 살펴보는 프로젝트를 진행하고 있습니다.

- **Database-based Service**  
  데이터베이스와 API를 연결해 데이터를 조회하고 활용하는 간단한 기능을 만들어 보고 있습니다.

- **RAG & Agent Experiment**  
  문서 검색, 데이터 조회, 요약 기능을 연결해 데이터 활용을 돕는 간단한 Agent 흐름을 실험하고 있습니다.
