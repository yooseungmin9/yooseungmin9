## 🌱 Interests

- **Data Analysis** — 데이터를 정리하고, 패턴을 찾고, 시각화하는 과정
- **AI Agents & RAG** — 검색, Tool Calling, 외부 API를 연결하는 Agent 구조
- **Workflow Automation** — 반복적인 정보 수집과 정리 과정을 효율화하는 방법
- **Backend & Data Engineering** — API, 데이터베이스, 데이터 처리 흐름을 연결하는 과정

## 📖 Learning Focus

> 작은 기능을 직접 구현하며 데이터와 Agent 기반 시스템의 흐름을 익히고 있습니다.

- **Data Collection & ETL**  
  API, 웹 스크래핑, 파일, 공개 데이터 등 여러 소스의 정보를 수집하고 정제·저장하는 과정

- **RAG & Knowledge Search**  
  문서와 지식 데이터를 검색하고, 관련 정보를 바탕으로 답변을 생성하는 방식

- **Agent Workflow**  
  검색, 데이터 조회, 요약 등 필요한 작업을 도구와 연결해 수행하는 흐름

- **Responsible Automation**  
  보안, 권한, 검토 절차를 고려하며 사람의 판단을 보조하는 자동화 방식
